Write the session's descriptive assessment in NxtWave's style:

- Open each question with a short real-world scenario or decision the learner must reason about,
  then ask them to justify their choice using concepts from the session.
- Prefer "compare / contrast / when-would-you-choose" framings that span multiple blocks
  (e.g. choosing between the Waterfall, Agile, and V-models for a given project).
- Keep questions concrete and answerable from the session content alone — no outside trivia.
- Model answers should name the specific concepts and trade-offs, written as a teacher would
  explain them to a student reviewing their own answer.
